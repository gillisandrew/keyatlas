# Ghostty Keyboard Shortcuts

Default keyboard shortcuts for Ghostty on macOS.

## Window Management

| Action | Shortcut |
|--------|----------|
| New window | <kbd>⌘</kbd><kbd>N</kbd> |
| Close window | <kbd>⌘</kbd><kbd>⇧</kbd><kbd>W</kbd> |
| Close all windows | <kbd>⌘</kbd><kbd>⇧</kbd><kbd>⌥</kbd><kbd>W</kbd> |
| Toggle fullscreen | <kbd>⌘</kbd><kbd>↵</kbd> or <kbd>⌘</kbd><kbd>⌃</kbd><kbd>F</kbd> |
| Quit application | <kbd>⌘</kbd><kbd>Q</kbd> |

## Tab Management

| Action | Shortcut |
|--------|----------|
| New tab | <kbd>⌘</kbd><kbd>T</kbd> |
| Close tab | <kbd>⌘</kbd><kbd>W</kbd> |
| Previous tab | <kbd>⌘</kbd><kbd>⇧</kbd><kbd>[</kbd> |
| Next tab | <kbd>⌘</kbd><kbd>⇧</kbd><kbd>]</kbd> |
| Go to tab 1–8 | <kbd>⌘</kbd><kbd>1</kbd> – <kbd>⌘</kbd><kbd>8</kbd> |
| Go to last tab | <kbd>⌘</kbd><kbd>9</kbd> |

## Split Management

| Action | Shortcut |
|--------|----------|
| New split (right) | <kbd>⌘</kbd><kbd>D</kbd> |
| New split (down) | <kbd>⌘</kbd><kbd>⇧</kbd><kbd>D</kbd> |
| Focus previous split | <kbd>⌘</kbd><kbd>[</kbd> |
| Focus next split | <kbd>⌘</kbd><kbd>]</kbd> |
| Focus split up | <kbd>⌘</kbd><kbd>⌥</kbd><kbd>↑</kbd> |
| Focus split down | <kbd>⌘</kbd><kbd>⌥</kbd><kbd>↓</kbd> |
| Focus split left | <kbd>⌘</kbd><kbd>⌥</kbd><kbd>←</kbd> |
| Focus split right | <kbd>⌘</kbd><kbd>⌥</kbd><kbd>→</kbd> |
| Toggle split zoom | <kbd>⌘</kbd><kbd>⇧</kbd><kbd>↵</kbd> |
| Resize split up | <kbd>⌘</kbd><kbd>⌃</kbd><kbd>↑</kbd> |
| Resize split down | <kbd>⌘</kbd><kbd>⌃</kbd><kbd>↓</kbd> |
| Resize split left | <kbd>⌘</kbd><kbd>⌃</kbd><kbd>←</kbd> |
| Resize split right | <kbd>⌘</kbd><kbd>⌃</kbd><kbd>→</kbd> |
| Equalize splits | <kbd>⌘</kbd><kbd>⌃</kbd><kbd>=</kbd> |

## Copy & Paste

| Action | Shortcut |
|--------|----------|
| Copy | <kbd>⌘</kbd><kbd>C</kbd> |
| Paste | <kbd>⌘</kbd><kbd>V</kbd> |

## Scrollback & Navigation

| Action | Shortcut |
|--------|----------|
| Scroll to top | <kbd>⌘</kbd><kbd>Home</kbd> |
| Scroll to bottom | <kbd>⌘</kbd><kbd>End</kbd> |
| Scroll page up | <kbd>⌘</kbd><kbd>Page Up</kbd> |
| Scroll page down | <kbd>⌘</kbd><kbd>Page Down</kbd> |
| Jump to previous prompt | <kbd>⌘</kbd><kbd>↑</kbd> |
| Jump to next prompt | <kbd>⌘</kbd><kbd>↓</kbd> |
| Clear screen | <kbd>⌘</kbd><kbd>K</kbd> |
| Write scrollback to file (paste) | <kbd>⌘</kbd><kbd>⇧</kbd><kbd>J</kbd> |
| Write scrollback to file (open) | <kbd>⌘</kbd><kbd>⇧</kbd><kbd>⌥</kbd><kbd>J</kbd> |

## Font Size

| Action | Shortcut |
|--------|----------|
| Increase font size | <kbd>⌘</kbd><kbd>+</kbd> / <kbd>⌘</kbd><kbd>=</kbd> |
| Decrease font size | <kbd>⌘</kbd><kbd>-</kbd> |
| Reset font size | <kbd>⌘</kbd><kbd>0</kbd> |

## Configuration

| Action | Shortcut |
|--------|----------|
| Open config | <kbd>⌘</kbd><kbd>,</kbd> |
| Reload config | <kbd>⌘</kbd><kbd>⇧</kbd><kbd>,</kbd> |

## Inspector

| Action | Shortcut |
|--------|----------|
| Toggle inspector | <kbd>⌘</kbd><kbd>⌥</kbd><kbd>I</kbd> |

---

> All keybindings can be customized via the `keybind` option in your configuration file.
